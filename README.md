# REST client for B24

## Project setup
1. Install packages
```
npm install
```

2. Create CLI **bx-auth** 
```
sudo npm link
```

## Auth profile actions:
1. Create new
```
bx-auth new
```

3. Show list
```
bx-auth list
```

2. Delete
```
bx-auth delete (--name <name>)
```

3. Change
```
bx-auth change (--name <name>)
```

## Run workspace
```
npm start
```
